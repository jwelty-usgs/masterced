Conservation Efforts Database
=============================
Application Framework
* Python 2.7.7
* Django 1.11.15
* ArcGIS 10.2
* MySQL 5.6
* Apache 2.2

Required Packages
* Apache mod_wsgi
* ArcGIS Desktop 10.2 arcpy
* see "requirements.txt" for Python packages

APIs Used
* ScienceBase 2.x for documents and data persistence
* AGOL for footprinter
